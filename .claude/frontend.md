# 🌐 Next.js Web App (Auth + Weather + Nearby Parks)

As a expert in frontend build web application built using **Next.js (App Router)** with:

- Login page with
    - 🔐 Google Authentication
- On login it should have browser Location Access
- Dashboard page to show weather details, parks recommendation via OpenStreetMap api
- 🌦 Weather Data (OpenWeather API)
- 🌳 Nearby Parks (OpenStreetMap / Overpass API)

---

## 🚀 Tech Stack

- Next.js (App Router)
- NextAuth.js (Google Auth)
- Tailwind CSS (optional)
- OpenWeather API
- OpenStreetMap (Overpass API)
- Geolocation API (Browser)
